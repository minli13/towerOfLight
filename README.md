Tower Of Light

CONTROLS:
- WASD: Move character
- E/Space: Interact with NPCs/objects
- Tab: Open tool ring
- I: Open inventory
- Mouse: Click on UI elements

HOW TO RUN:
1. Extract the ZIP file here: [Tower Of Light](https://drive.google.com/file/d/1jObhA5jWgl3u4N7BS_JmznVvzV6VfeBb/view?usp=sharing)
2. Run "TowerOfLight.exe" (Windows)
   OR "TowerOfLight.app" (Mac)
   OR "TowerOfLight.x86_64" (Linux)
